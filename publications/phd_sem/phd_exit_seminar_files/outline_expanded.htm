Properties of OFDM
Advantages
Strong multipath tolerance
Each sub-carrier is independent
Spectrally Efficient
Square spectrum
Allows QAM and PSK subcarrier modulation
Disadvantages
Sensitive to frequency errors (ferror<1% of sub carrier spacing)
Requires linear amplification
Has a high peak to average power ratio (9-12dB)

Adaptive Modulation
Advantages
Ensures optimal matching loading of data
Greatly reduces BER (2,000 x)
Reduces need for Forward Error Correction
Reduces the required link margin by 12-16 dB
Disadvantages
Requires tracking of the radio channel (200-1000 times/sec) (This is achievable)
Requires low delay two-way communications
Mistakes in reallocation could cause problems

Adaptive User Allocation
Measure the channel response to each user.
Allocate a bandwidth for each based on the channel path loss.
Weak users can only support a small bandwidth
Allocate worst path loss users first.
Allocate them subcarriers with the highest SNR for that user.
Avoids the nulls in the spectrum
Continue until all users are allocated

Adaptive User Allocation

Modulation at each frequency

Adaptive User Allocation
Advantages
Utilises Frequency Selective Fading to increase received power
Average received signal power is improved by 3-5 dB
Avoids nulls in the spectrum
Disadvantages
Requires complete knowledge of entire system bandwidth to every user
Large information overhead, more than adaptive modulation
