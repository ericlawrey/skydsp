Digital data is transferred by each subcarrier sending an IQ vector in each data symbol. These correspond the red crosses in the above figures and is achieved by setting the amplitude and phase of the subcarrier. The amplitude corresponds to the distance from the origin while the phase corresponds to the angle of the vector. Each IQ vector is mapped to a group of transmitted bits (shown in white next to the crosses in 16 PSK). This mapping is known by both the transmitter and receiver.
The signal is decoded by the receiver by picking the closest known transmitter IQ vector to the received vector. The received data is generated by using a lookup table to find the corresponding bits of information for the received vector.
Two example subcarrier modulation schemes are shown above, 16 PSK (Phase Shift Keying)  and 16 QAM (Quadrate Amplitude Modulation). Each of these schemes has 16 possible transmitter vectors, and so can transmit 4 bits (2^4 = 16) for each symbol transmitted.
