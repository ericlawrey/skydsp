OFDM signals can be easily generated using an Inverse Fast Fourier Transform (IFFT). The IFFT converts a set of frequency domain complex vectors, which correspond to the amplitude and phase of each subcarrier, into the time domain. Guard subcarriers on either side of the OFDM signal are set to zero amplitude, so that the signal generated doesn’t use the entire nyquist bandwidth. This allows for practical anti-aliasing filters to be used when converting the OFDM signal from a sampled waveform to an analog one for transmission. Typically 80% of the nyquist bandwidth is used, i.e. the outer upper and lower 10% of the IFFT subcarriers are set to zero.
After the signal is converted to the time domain a guard period is usually added to the start of each symbol. The function of this is described in a later slide.
